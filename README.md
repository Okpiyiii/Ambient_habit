## 📖 Overview

Most habit trackers feel like a chore. **Ambient Habit** shifts the focus from "checking boxes" to "creating flow." It combines habit tracking with an ambient aesthetic to help you focus on your growth without the digital noise.

### ✨ Key Features

- **🧘 Minimalist UI:** A clean, soothing interface that reduces cognitive load.
- **📊 Progress Tracking:** Visualize your streaks and long-term consistency.
- **🎵 Ambient Focus:** (Optional: Integrated lo-fi or nature sounds to keep you in the zone).
- **📱 Responsive Design:** Track your habits seamlessly on desktop or mobile.
- **🌙 Dark Mode:** Easy on the eyes for late-night reflection and morning planning.

## 🚀 Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.)
- (Optional) [Node.js](https://nodejs.org/) if you plan to run a local dev server.

### Installation

1. **Clone the repo**
   ```bash

2.Navigate to the directory
Bash

3.Open the project
